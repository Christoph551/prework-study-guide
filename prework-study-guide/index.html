Git Section - Communication with your local computer.

  1. git status: checks what branch we are currently on
  2. git checkout -b branch-name: creates a new branch and switches to it
  3. git add -A: Commits all files in the working branch
  4. git commit -m "short descriptive text": Associates a message with the commit.
